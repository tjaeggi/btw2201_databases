## UE04-02 Suche nach Artist

!!! example "UE04-02-Suche nach Artist"

    Schreibe Sie ein Script, welches nach einem beliebigen Artisten fragt. Dazu können Sie die Funktion input() verwenden. Sobald Sie den Artist eingelesen haben, holen Sie die Anzahl von Songs zu diesem Artist. Zum Beispiel:

    Das Resultat könnte wie folgt aussehen:
    
    ```
    Which band to look for? U2
    U2 has 135 songs

    Which band to look for? Foo Fighters
    Foo Fighters has 44 songs

    Which band to look for? xyz
    Sadly the band has not been found.
    ```
